# Upstream Exploration & Production (E&P) Portfolio Analytics

---

## 🎯 Portfolio Purpose & Goal
This project was conceptualized and developed as a dedicated **Technical Excellence & Coordination Internship**. The core objective is to practically simulate the analytical, data engineering, and monitoring responsibilities carried out within an upstream development environment, specifically mapping onto the core competencies required for:

* **Target Company:** PT Pertamina Hulu Energi (PHE)
* **Target Function:** D&P Technical Excellence & Coordination (Reservoir & Production)

Through this portfolio, the developer showcases data-driven readiness to handle asset screening operations, understand financial-technical constraints in reservoir/production management, and manage upstream relational data structures efficiently.

---

## 📌 Project Overview
As part of regional asset optimization, this repository demonstrates a data-driven approach to evaluating the economic viability, cost structure, and investment feasibility of a **60-field exploration and production (E&P) asset portfolio** split across Onshore and Offshore environments. 

The analytical workflow transitions from raw data preprocessing to multi-dimensional data engineering (SQL) and dynamic business reporting (Excel Executive Dashboard) to assist senior stakeholders in making data-driven **Final Investment Decisions (FID)**.

> ⚠️ **Data Disclaimer:** All data utilized in this project is **100% synthetic and generated via AI** for portfolio demonstration and technical assessment purposes only. It does not reflect any real-world asset metrics or confidential corporate data of any specific entity.
